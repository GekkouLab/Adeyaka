# Adeyaka
## 中文脚本插件

<div align="center">
   <img width="240" src="" alt="logo"></div>

----

**Adeyaka** 是一个主要使用 **Kotlin** 及 **Java** 编写的 **中文脚本插件**， 为 **Bukkit** 服务器 提供 **中文书写的** 脚本功能 *(虽然实际上底层是通用的..)*

这个项目的名字来源于日语 艶やか
